# Integrating LangChain with ChatGPT

In this module we will explore how to integrate LangChain with ChatGPT to build more advanced applications.

## Introduction to Chains

Chains are a core concept in LangChain that allow you to link together multiple components (like LLMs, prompt templates, and tools) to create more complex workflows. By chaining these components, you can build applications that can perform multi-step reasoning, data retrieval, and more.
